about

This gem provides a clean and powerful interface for interacting with the Flickr API. It is designed to be simple to begin using, but supports advanced usage for photo uploads and integration into a web application. To get started, you just need to configure it with a valid API key.

features

Fleakr supports:

  • Retrieval of resources (e.g users, contacts, etc..) with finders or through associations
  • Single-user authentication via Mobile authentication mode
  • Multi-user authentication via Web authentication mode
  • Individual and bulk uploading of photos
  • Individual and bulk downloading of photos to a destination path

installation

Fleakr is available via RubyGems:

{% highlight bash %} $ gem install fleakr {% endhighlight %}

Or you can grab a prerelease version from Github:

{% highlight bash %} $ git clone git://github.com/reagent/fleakr.git $ cd fleakr $ rake gem $ gem install pkg/fleakr-.gem {% endhighlight %}

basic usage

To get started, require the gem and set your API key:

{% highlight ruby %} require 'rubygems' require 'fleakr' Fleakr.api_key = 'my_api_key' {% endhighlight %}

Most associations can be found from a single user account:

{% highlight ruby %} user = Fleakr.user('the_reagent') user.photos user.photos.first.tags user.photos.first.comments user.sets user.sets.first.photos user.sets.first.save_to('/tmp', :original) photo = user.sets.first.photos.first photo.url photo.large.url photo.large.save_to('/tmp', :large) user.contacts {% endhighlight %}

See the documentation for more information.
