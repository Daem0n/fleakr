Fleakr

A small, yet powerful, gem to interface with Flickr photostreams

About

This gem provides a clean Ruby interface for interacting with the Flickr REST API. I designed it to be simple to use - you only need to configure the API key to start using it. If you want to make signed or authenticated calls, you can do that as well. See the Documentation section for more information.

Features

  • Single-user authentication support via Mobile authentication mode
  • Multi-user authentication support via Web authentication mode
  • Single and multiple file uploading support
  • Download individual photos or sets to a destination path

Installation

If you prefer stability in your life, install the latest version via RubyGems:

$ sudo gem install fleakr

If you prefer to bleed, grab the latest updates from the repository:

$ git clone git://github.com/reagent/fleakr.git
$ cd fleakr
$ rake gem
$ gem install pkg/fleakr-<version>.gem

Usage / Examples

To get started, require the gem and set your API key:

  require 'rubygems'
  require 'fleakr'
  
  Fleakr.api_key = 'my_api_key'

Most associations can be found from a single user account:

  user = Fleakr.user('the_reagent')
  user.photos
  user.photos.first.tags
  user.photos.first.comments
  
  user.sets
  user.sets.first.photos
  user.sets.first.save_to('/tmp', :original)
  
  photo = user.sets.first.photos.first
  photo.url
  photo.large.url
  photo.large.save_to('/tmp', :large)
  
  user.contacts

See the documentation for more information.
